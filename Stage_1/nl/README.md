---

## 🚀 Hoe begin ik?

Om dit lab interactief te ervaren, moet **Livebook** op je computer geïnstalleerd zijn.

1. **Installeer Livebook:** Download het op [livebook.dev](https://livebook.dev).
2. **Open een Lab:** Je kunt door de onderstaande mappen bladeren en op de knop **"Run in Livebook"** in elk bestand klikken, of de URL rechtstreeks in je Livebook-sessie importeren.

---

## 📚 Coding5s Methodologie (5 Stages)

Onze aanpak is gebaseerd op **Gecontroleerde Cognitieve Frictie**. Elk lab begeleidt je door:

| Stage | Naam | Doel |
|-------|--------|----------|
| **1** | Practice | Code schrijven vanaf nul |
| **2** | Debug | Opzettelijke logische fouten vinden |
| **3** | Complete | Architecturale gaten opvullen |
| **4** | Refactor | Bestaande code optimaliseren |
| **5** | Extend | Nieuwe functionaliteiten toevoegen |

**Deze repository bevat Stage 1 (gratis).** Stages 2-5 zijn beschikbaar in de premium versie.

---

## 🧠 De Socratische Mentor (AI)

In elk bestand vind je een **Mentor Prompt**. Dit is geen gewone chatbot; het is een simulator voor elite-mentoren die:

- **Je niet direct het antwoord geeft** (in het begin)
- Je vragen stelt zodat je zelf de fout ontdekt
- Analogieën uit de echte wereld gebruikt om BEAM-concepten uit te leggen
- Je helpt om "Vibe Coding" te elimineren en echte technische intuïtie op te bouwen
- Stel hem vragen en hij geeft je antwoorden op Senior Engineer-niveau.

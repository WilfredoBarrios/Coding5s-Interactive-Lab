## 🚀 Wie fange ich an?

Um dieses Labor interaktiv zu erleben, muss **Livebook** auf deinem Computer installiert sein.

1. **Installiere Livebook:** Lade es unter [livebook.dev](https://livebook.dev) herunter.
2. **Öffne ein Labor:** Du kannst die untenstehenden Ordner durchsuchen und auf die Schaltfläche **"Run in Livebook"** in jeder Datei klicken oder die URL direkt in deine Livebook-Sitzung importieren.

## 📚 Die Coding5s-Methodik (5 Phasen)

Unser Ansatz basiert auf **"Controlled Cognitive Friction"** (Gezielte kognitive Reibung). Jedes Labor führt dich durch:

| Phase | Name | Ziel |
|-------|--------|----------|
| **1** | Practice | Code von Grund auf neu schreiben |
| **2** | Debug | Absichtliche logische Fehler finden |
| **3** | Complete | Architektonische Lücken füllen |
| **4** | Refactor | Bestehenden Code optimieren |
| **5** | Extend | Neue Funktionalitäten hinzufügen |

**Dieses Repository enthält Phase 1 (kostenlos).** Die Phasen 2-5 sind in der Premium-Version verfügbar.

## 🧠 Der Sokratische Mentor (KI)

In jeder Datei findest du einen **Mentor-Prompt**. Dies ist kein gewöhnlicher Chatbot, sondern ein Simulator für Elite-Mentoren, der:

- **Dir die Antwort nicht direkt gibt** (anfangs)
- Dir Fragen stellt, damit du den Fehler selbst entdeckst
- Analogien aus der realen Welt verwendet, um BEAM-Konzepte zu erklären
- Dir hilft, "Vibe Coding" zu vermeiden und eine echte technische Intuition aufzubauen
- Stelle ihm Fragen, und er wird dir Antworten auf Senior-Engineer-Niveau geben.

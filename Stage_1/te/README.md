## 📚 Coding5s పద్ధతి (5 దశలు)

మా విధానం **"నియంత్రిత సంజ్ఞానాత్మక ఘర్షణ" (Controlled Cognitive Friction)** పై ఆధారపడి ఉంటుంది. ప్రతి ల్యాబ్ మిమ్మల్ని ఈ దశల ద్వారా నడిపిస్తుంది:

| దశ | పేరు | లక్ష్యం |
|-------|--------|----------|
| **1** | Practice | మొదటి నుండి కోడ్ రాయడం |
| **2** | Debug | ఉద్దేశపూర్వక లాజికల్ ఎర్రర్‌లను కనుగొనడం |
| **3** | Complete | ఆర్కిటెక్చరల్ లోపాలను పూరించడం |
| **4** | Refactor | ఉన్న కోడ్‌ను ఆప్టిమైజ్ చేయడం |
| **5** | Extend | కొత్త ఫీచర్లను జోడించడం |

**ఈ రిపోజిటరీలో దశ 1 (ఉచితం) ఉంది.** దశలు 2-5 ప్రీమియం వెర్షన్‌లో అందుబాటులో ఉన్నాయి.

## 🧠 సోక్రటిక్ మెంటర్ (AI)

ప్రతి ఫైల్ లోపల మీరు ఒక **"మెంటర్ ప్రాంప్ట్" (Mentor Prompt)** ను కనుగొంటారు. ఇది సాధారణ చాట్‌బాట్ కాదు; ఇది ఒక ఎలైట్ మెంటర్ సిమ్యులేటర్:

- ఇది మీకు నేరుగా సమాధానం ఇవ్వదు (మొదట్లో)
- మీ తప్పును మీరే కనుగొనేలా మిమ్మల్ని ప్రశ్నలు అడుగుతుంది
- BEAM కాన్సెప్ట్‌లను వివరించడానికి నిజ జీవిత ఉదాహరణలను ఉపయోగిస్తుంది
- మీ ఇంజనీరింగ్ అంతర్ దృష్టిని (Intuition) నిర్మించడంలో సహాయపడుతుంది
- దీనిని ప్రశ్నలు అడగండి, ఇది మీకు సీనియర్ ఇంజనీర్ స్థాయి సమాధానాలను ఇస్తుంది.

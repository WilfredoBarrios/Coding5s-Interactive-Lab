## 🚀 Comment commencer ?

Pour expérimenter ce laboratoire de manière interactive, vous devez avoir **Livebook** installé sur votre ordinateur.

1. **Installez Livebook :** Téléchargez-le sur [livebook.dev](https://livebook.dev).
2. **Ouvrez un Laboratoire :** Parcourez les dossiers ci-dessous et cliquez sur le bouton **"Run in Livebook"** à l'intérieur de chaque fichier, ou importez directement l'URL dans votre session Livebook.

---

## 📚 Méthodologie Coding5s (5 Stages)

Notre approche repose sur la **Friction Cognitive Contrôlée**. Chaque laboratoire vous guidera à travers :

| Stage | Nom | Objectif |
|-------|--------|----------|
| **1** | Practice | Écrire du code à partir de zéro |
| **2** | Debug | Trouver des erreurs logiques intentionnelles |
| **3** | Complete | Combler les lacunes architecturales |
| **4** | Refactor | Optimiser le code existant |
| **5** | Extend | Ajouter de nouvelles fonctionnalités |

**Ce dépôt contient l'Étape 1 (gratuite).** Les Étapes 2 à 5 sont disponibles dans la version premium.

---

## 🧠 Le Mentor Socratique (IA)

Dans chaque fichier, vous trouverez un **Prompt de Mentor**. Ce n'est pas un chatbot ordinaire ; c'est un simulateur de mentor d'élite qui :

- **Ne vous donnera pas la réponse directement** (au début)
- Vous posera des questions pour que vous découvriez l'erreur par vous-même
- Utilise des analogies du monde réel pour expliquer les concepts de la BEAM
- Vous aide à éliminer le "Vibe Coding" et à bâtir une véritable intuition d'ingénieur
- Posez-lui des questions, il vous donnera des réponses de niveau Senior.

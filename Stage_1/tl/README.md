## 🚀 Paano magsimula?

Para maranasan ang laboratoryo na ito nang interactive, kailangan mong may nakainstall na **Livebook** sa iyong computer.

1. **I-install ang Livebook:** I-download ito sa [livebook.dev](https://livebook.dev).
2. **Buksan ang Lab:** Maaari mong i-browse ang mga folder sa ibaba at i-click ang button na **"Run in Livebook"** sa loob ng bawat file, o i-import ang URL nang direkta sa iyong Livebook session.

---

## 📚 Metodolohiya ng Coding5s (5 Stages)

Ang aming diskarte ay nakabase sa **Controlled Cognitive Friction**. Gabay mo ang bawat lab sa pamamagitan ng:

| Stage | Pangalan | Layunin |
|-------|--------|----------|
| **1** | Practice | Magsulat ng code mula sa simula |
| **2** | Debug | Humanap ng mga sadyang logical error |
| **3** | Complete | Punan ang mga gaps sa architecture |
| **4** | Refactor | I-optimize ang kasalukuyang code |
| **5** | Extend | Magdagdag ng mga bagong functionality |

**Ang repository na ito ay naglalaman ng Stage 1 (libre).** Ang Stages 2-5 ay available sa premium version.

## 🧠 Ang Socratic Mentor (AI)

Sa loob ng bawat file, makakakita ka ng isang **Mentor Prompt**. Hindi ito simpleng chatbot; ito ay isang elite mentor simulator na:

- **Hindi ibibigay ang sagot nang direkta** (sa simula)
- Magtatanong sa iyo para ikaw mismo ang makatuklas ng error
- Gagamit ng mga real-world analogy para ipaliwanag ang mga BEAM concepts
- Tutulungan kang alisin ang "Vibe Coding" at bumuo ng tunay na engineering intuition
- Magtanong ka lang at bibigyan ka niya ng mga sagot na level Senior Engineer.
